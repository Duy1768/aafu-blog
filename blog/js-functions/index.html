Hàm trong JavaScript

Hàm là một khối mã có thể tái sử dụng và thực hiện một tác vụ cụ thể trong chương trình. Trong JavaScript, hàm là một đối tượng có thể nhận dữ liệu đầu vào và trả về kết quả.

Sử dụng hàm giúp:

  • Tái sử dụng mã nguồn.
  • Tổ chức code gọn gàng, dễ đọc.
  • Dễ bảo trì và mở rộng.

1. Hàm trong JavaScript là gì?

Hàm là một khối mã được đặt tên, có thể tái sử dụng và thực thi nhiều lần. Nó có thể chứa các lệnh và tham số để thực hiện một tác vụ nào đó.

2. Cách định nghĩa hàm

Cách phổ biến nhất là sử dụng từ khóa function:

function myFunction() {
  console.log("Hello, world!");
}

myFunction là tên hàm.

Khối { … } chứa các lệnh cần thực thi.

2.1 Hàm có tham số

Tham số là các biến được truyền vào hàm để sử dụng trong quá trình thực thi.

function greet(name, age) {
  console.log("Hello " + name + ", you are " + age + " years old.");
}

greet("John", 30); // Output: Hello John, you are 30 years old.

name và age là tham số.

Khi gọi hàm, truyền các giá trị cụ thể vào.

2.2 Giá trị trả về

Hàm có thể trả về một giá trị bằng từ khóa return.

function add(x, y) {
  return x + y;
}

let sum = add(5, 10);
console.log(sum); // Output: 15

return x + y trả về tổng hai số.

Biến sum nhận giá trị trả về của hàm.

3. Cách sử dụng hàm

Để sử dụng hàm, bạn gọi tên hàm và truyền tham số nếu cần:

function introduce(name, age) {
  console.log("My name is " + name + " and I am " + age + " years old.");
}

introduce("Alice", 25); 
// Output: My name is Alice and I am 25 years old.

Gọi hàm nhiều lần với các tham số khác nhau để tái sử dụng code.

Có thể kết hợp hàm với các biểu thức, vòng lặp hoặc điều kiện.
