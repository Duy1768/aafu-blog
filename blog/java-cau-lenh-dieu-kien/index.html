Câu lệnh điều kiện trong Java

Trong lập trình Java, đôi khi chúng ta cần thực hiện các hành động dựa trên điều kiện nhất định. Những câu lệnh này được gọi là câu lệnh điều kiện hay câu lệnh rẽ nhánh.

Trong Java, có hai cách chính để xử lý điều kiện:

  • if-else
  • switch-case

1. Câu lệnh if-else

a. Câu lệnh if

  • Thực thi code bên trong nếu điều kiện trả về true
int number = 10;

if (number <= 10) {
    System.out.printf("number nhỏ hơn hoặc bằng %d", number);
}
// Output: number nhỏ hơn hoặc bằng 10

b. Câu lệnh if-else

Nếu điều kiện if sai, thực thi else

int number = 11;

if (number <= 10) {
    System.out.println("number nhỏ hơn hoặc bằng 10");
} else {
    System.out.println("number lớn hơn 10");
}
// Output: number lớn hơn 10

c. Else-if nhiều điều kiện

float point = 7;

if (point < 5) {
    System.out.println("Dưới trung bình");
} else if (point >= 5 && point < 6.5) {
    System.out.println("Trên trung bình");
} else if (point >= 6.5 && point <= 8.5) {
    System.out.println("Khá");
} else {
    System.out.println("Giỏi");
}
// Output: Khá

d. If-else lồng nhau

int number = 7;

if (number <= 10) {
    if (number % 2 == 0) {
        System.out.println("Số chẵn");
    } else {
        System.out.println("Số lẻ");
    }
} else {
    System.out.println("Số lớn hơn 10");
}

2. Câu lệnh switch-case

Kiểm tra xem một biến có bằng với giá trị nào không

Dùng cho các kiểu dữ liệu nguyên thủy, enum, String, hoặc wrapper class

Cú pháp:

switch (variable) {
    case value1:
        // code
        break;
    case value2:
        // code
        break;
    default:
        // code
        break;
}

Ví dụ đơn giản:

int number = 9;

switch(number){
    case 10:
        System.out.println("Xuất sắc");
        break;
    default:
        System.out.println("Cố gắng");
        break;
}
// Output: Cố gắng

Gom nhóm case:

int number = 3;

switch(number){
    case 1:
    case 2:
    case 3:
    case 4:
        System.out.println("Nhỏ hơn 5");
        break;
    default:
        System.out.println("Lớn hơn 5");
        break;
}
// Output: Nhỏ hơn 5

Switch-case kết hợp if-else
int month = 2;
int year = 2020;
int numDays = 0;

switch (month) {
    case 1: case 3: case 5: case 7: case 8: case 10: case 12:
        numDays = 31;
        break;
    case 4: case 6: case 9: case 11:
        numDays = 30;
        break;
    case 2:
        if (((year % 4 == 0) && !(year % 100 == 0)) || (year % 400 == 0)) {
            numDays = 29;
        } else {
            numDays = 28;
        }
        break;
    default:
        System.out.println("Invalid month.");
        break;
}
System.out.println("Number of Days = " + numDays);
// Output: Number of Days = 29

3. Khi nào nên dùng if-else hoặc switch-case

Switch-case: khi điều kiện cần check đơn giản, nhiều hơn 3 trường hợp, hoặc so sánh bằng

If-else: khi điều kiện phức tạp, hoặc kiểm tra không phải so sánh bằng
