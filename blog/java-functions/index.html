Hàm trong Java (Phần 1)

Trong ngôn ngữ lập trình Java, một hàm (function) là một khối code chứa một tập hợp các câu lệnh được đặt tên và thực hiện một nhiệm vụ cụ thể. Hàm giúp tái sử dụng code, tạo cấu trúc rõ ràng và dễ quản lý trong các ứng dụng phức tạp.

1. Ví dụ giới thiệu về hàm

public class HelloWorld {
    // Hàm đơn giản in ra "Hello, world!"
    public static void printHello() {
        System.out.println("Hello, world!");
    }

    public static void main(String[] args) {
        // Gọi hàm printHello
        printHello();
    }
}

Trong ví dụ trên:

Hàm printHello được khai báo với public static void.

Khi gọi hàm, nó in ra “Hello, world!”.

Hàm có thể được gọi từ bất kỳ đâu trong chương trình.

2. Định nghĩa một hàm trong Java

Công thức tổng quát:

[Quyền truy cập] [Kiểu dữ liệu trả về] [Tên hàm]([Danh sách tham số]) {
    // Khối lệnh (Body)
    [Các câu lệnh thực thi]
    [return Giá_trị;] // nếu có
}

Các thành phần chính:

Quyền truy cập: public, private, protected, hoặc không có modifier.

Kiểu dữ liệu trả về: void nếu không trả về giá trị, hoặc kiểu dữ liệu cụ thể.

Tên hàm: tên hàm theo quy tắc Java.

Danh sách tham số: kiểu dữ liệu và tên tham số.

Khối lệnh (Body): logic của hàm.

Lệnh return: chỉ dùng nếu kiểu trả về không phải void.

Ví dụ tính tổng hai số:

public int addNumbers(int a, int b) {
    int sum = a + b;
    return sum;
}

3. Gọi hàm

Trong cùng một lớp:

int sum = addNumbers(5, 3);

Từ một lớp khác (không static):

OtherClass obj = new OtherClass();
int result = obj.someMethod(argument1, argument2);

Hàm tĩnh (static):

int result = MyClass.staticMethod(argument1, argument2);

4. Hàm void và hàm trả về giá trị

Hàm không trả về (void)

public void printHello() {
    System.out.println("Hello, World!");
}

Hàm có giá trị trả về
public int add(int a, int b) {
    return a + b;
}

5. Ví dụ minh họa

Ví dụ 1: Tính ước chung lớn nhất (GCD) bằng phương pháp Euclid

public class GreatestCommonDivisorMethod {

    public static int findGCD(int num1, int num2) {
        if (num2 == 0) {
            return num1;
        } else {
            return findGCD(num2, num1 % num2);
        }
    }

    public static void main(String[] args) {
        int number1 = 48;
        int number2 = 18;

        int gcd = findGCD(number1, number2);
        System.out.println("Ước chung lớn nhất của " + number1 + " và " + number2 + " là " + gcd);
    }
}

Ví dụ 2: Tính tiền lương nhân viên

public class EmployeeSalaryCalculator {

    public static double calculateMonthlySalary(double hourlyWage, int hoursWorked) {
        int hoursInMonth = 160;
        double monthlySalary = hourlyWage * hoursWorked;

        if (hoursWorked > hoursInMonth) {
            int overtimeHours = hoursWorked - hoursInMonth;
            double overtimePay = overtimeHours * (hourlyWage * 1.5);
            monthlySalary += overtimePay;
        }

        return monthlySalary;
    }

    public static void main(String[] args) {
        String employeeName = "Nguyen Van A";
        double hourlyWage = 12.5;
        int hoursWorked = 176;

        double monthlySalary = calculateMonthlySalary(hourlyWage, hoursWorked);
        System.out.println("Nhân viên: " + employeeName);
        System.out.println("Tiền lương hàng tháng: $" + monthlySalary);
    }
}

Ví dụ 3: Thuật toán Kruskal – Tìm cây bao trùm tối thiểu

import java.util.*;

class Edge {
    int source, destination, weight;
    public Edge(int source, int destination, int weight) {
        this.source = source;
        this.destination = destination;
        this.weight = weight;
    }
}

class Graph {
    int vertices;
    List<Edge> edges;
    public Graph(int vertices) {
        this.vertices = vertices;
        edges = new ArrayList<>();
    }

    public void addEdge(int source, int destination, int weight) {
        edges.add(new Edge(source, destination, weight));
    }

    public List<Edge> kruskalMST() {
        List<Edge> mst = new ArrayList<>();
        edges.sort(Comparator.comparingInt(e -> e.weight));

        int[] parent = new int[vertices];
        for (int i = 0; i < vertices; i++) parent[i] = i;

        int edgeCount = 0, i = 0;
        while (edgeCount < vertices - 1) {
            Edge current = edges.get(i);
            int srcParent = find(parent, current.source);
            int destParent = find(parent, current.destination);

            if (srcParent != destParent) {
                mst.add(current);
                union(parent, srcParent, destParent);
                edgeCount++;
            }
            i++;
        }

        return mst;
    }

    private int find(int[] parent, int vertex) {
        if (parent[vertex] != vertex) return find(parent, parent[vertex]);
        return vertex;
    }

    private void union(int[] parent, int x, int y) {
        parent[find(parent, y)] = find(parent, x);
    }
}

public class KruskalExample {
    public static void main(String[] args) {
        Graph graph = new Graph(6);
        graph.addEdge(0, 1, 4);
        graph.addEdge(0, 2, 4);
        graph.addEdge(1, 2, 2);
        graph.addEdge(1, 3, 3);
        graph.addEdge(1, 4, 7);
        graph.addEdge(2, 3, 5);
        graph.addEdge(2, 4, 7);
        graph.addEdge(3, 4, 2);
        graph.addEdge(3, 5, 5);
        graph.addEdge(4, 5, 2);

        List<Edge> mst = graph.kruskalMST();
        for (Edge edge : mst) {
            System.out.println(edge.source + " - " + edge.destination + " : " + edge.weight);
        }
    }
}
