Các loại vòng lặp trong JavaScript

Vòng lặp là một thành phần quan trọng trong lập trình, giúp lặp lại các thao tác nhiều lần. Trong JavaScript, bạn có nhiều loại vòng lặp khác nhau từ cơ bản đến nâng cao. Bài viết này sẽ giới thiệu chi tiết từng loại và ví dụ minh họa.

1. For Loop

Vòng lặp for phổ biến nhất trong nhiều ngôn ngữ lập trình, dễ dàng kiểm soát biến đếm, điều kiện và bước nhảy.

Cú pháp:

for ([initialization]; [condition]; [final-expression]) {
  // block of code
}

Ví dụ:

for (var i = 0; i < 10; i++) {
  console.log(i);
}

Kết quả:

0 1 2 ... 9

2. While Loop

while thực thi khối lệnh khi điều kiện còn đúng.

Cú pháp:

while (condition) {
  // block of code
}

Ví dụ:

var i = 0;
while (i < 10) {
  console.log(i);
  i++;
}

Kết quả:

0 1 2 ... 9

3. Do…While

Khác với while, do…while luôn thực thi ít nhất một lần.

Cú pháp:

do {
  // block of code
} while (condition);

Ví dụ:

var i = 10;
do {
  console.log(i);
  i++;
} while (i > 10 && i < 12);

Kết quả:

10 11

4. forEach()

forEach lặp qua từng phần tử của mảng, thực thi function với mỗi phần tử.

Cú pháp:

arrayName.forEach(function(currentValue, index, array){
  // function body
});

Ví dụ:

var arr = [10, 20, "hi", , {}];
arr.forEach(function(item, index){
  console.log('arr[' + index + '] is ' + item);
});

Kết quả:

arr[0] is 10
arr[1] is 20
arr[2] is hi
arr[4] is [object Object]

5. Map

map tương tự forEach, nhưng tạo ra mảng mới dựa trên function.

Cú pháp:

var newArray = oldArray.map(function(currentValue, index, array){
  return modifiedValue;
});

Ví dụ:

var num = [1, 5, 10, 15];
var doubles = num.map(function(x) {
  return x * 2;
});
console.log(doubles);

Kết quả:

[2, 10, 20, 30]

6. For…in

Dùng để lặp qua thuộc tính của object. Có thể dùng với mảng, nhưng không đảm bảo thứ tự.

Cú pháp:

for (var key in object) {
  if (object.hasOwnProperty(key)) {
    // code block
  }
}

Ví dụ:

var obj = {a: 1, b: 2, c: 3};    
for (var prop in obj) {
  console.log('obj.' + prop + '=' + obj[prop]);
}

Kết quả:

obj.a=1
obj.b=2
obj.c=3

7. For…of

Ra mắt trong ES6, dùng để lặp các iterable như Array, String, Map, Set.

Cú pháp:

for (var value of iterable) {
  // block of code
}

Ví dụ với String:

var str = 'paul';
for (var char of str) {
  console.log(char);
}

Kết quả:

p a u l

Ví dụ với Map:

let itobj = new Map([['x', 0], ['y', 1], ['z', 2]]);
for (let [key, value] of itobj) {
  console.log(value);
}

Kết quả:

0 1 2
