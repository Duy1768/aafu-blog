Tối ưu câu lệnh điều kiện dễ đọc hơn trong JavaScript

Một trong những công cụ mạnh mẽ và được sử dụng nhiều nhất trong lập trình là câu lệnh điều kiện, giúp chạy các nhánh code chỉ khi điều kiện thỏa mãn. Trong JavaScript, có 3 loại câu lệnh điều kiện chính:

  1. if/else
  2. switch
  3. Biểu thức điều kiện (ternary operator)

Nếu không được tối ưu, các câu lệnh điều kiện dễ gây lỗi và khó đọc. Dưới đây là một số cách tối ưu giúp code của bạn gọn gàng và dễ bảo trì hơn.

1. Khái niệm cơ bản về conditionals

1.1 If / Else

if (condition) {
  // Chạy nếu condition đúng
} else {
  // Chạy nếu condition sai
}
Bạn có thể thêm nhiều điều kiện bằng else if:

if (condition1) {
  // code nếu thỏa mãn condition1
} else if (condition2) {
  // code nếu thỏa mãn condition2
} else {
  // code mặc định
}

1.2 Switch statement

Switch kiểm tra giá trị của một biểu thức (expression) và so khớp với các case. Nếu không có case nào thỏa mãn, bạn có thể dùng default:

switch (expression) {
  case value1:
    // code khi khớp value1
    break;
  case value2:
    // code khi khớp value2
    break;
  default:
    // code mặc định
    break;
}

Lưu ý: luôn dùng break hoặc return để ngắt thực thi case không thỏa mãn.

Ví dụ minh họa:

const name = "Jim";
switch(name) {
  case "Mark":
    console.log("Hello Mark");
    break;
  case "Jim":
    console.log("Hello Jim");
    break;
  default:
    console.log("Hello Stranger");
}

1.3 Biểu thức điều kiện (Ternary operator)

Một cách ngắn gọn để gán giá trị dựa trên điều kiện:

const myValue = condition ? valueIfTrue : valueIfFalse;

2. Best Practices cho conditionals

2.1 Dùng ternary thay vì if để gán giá trị

Không nên:

let myVal;
if (condition) {
  myVal = valueIfTrue;
} else {
  myVal = valueIfFalse;
}

Nên dùng:

const myVal = condition ? valueIfTrue : valueIfFalse;

2.2 Tránh lồng if/else quá nhiều

Thay vì:

let result;
if (condition) {
  result = result_v1;
} else if (condition2) {
  result = result_v2;
} else {
  result = default_result;
}
return result;

Bạn có thể viết gọn hơn bằng cách early return:

if (condition) return result_v1;
if (condition2) return result_v2;
return default_result;

2.3 Sử dụng switch khi có nhiều trường hợp

Switch giúp code dễ đọc hơn khi có nhiều điều kiện dựa trên cùng một biến:

switch(expression) {
  case "one":
    return result_v1;
  case "two":
    return result_v2;
  default:
    return default_result;
}
