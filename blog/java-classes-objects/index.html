Lớp và đối tượng trong Java

Trong lập trình hướng đối tượng (OOPs) trong Java, chúng ta thiết kế chương trình bằng cách sử dụng lớp (class) và đối tượng (object).

1. Đối tượng (Object)

Một đối tượng là một thực thể có trạng thái và hành vi. Ví dụ: máy pha cà phê, xe đạp, quạt,…

Các đặc điểm chính của đối tượng:

  • Trạng thái (State): đại diện cho dữ liệu của đối tượng (ví dụ: màu sắc, tên, giá trị).
  • Hành vi (Behavior): đại diện cho các chức năng mà đối tượng có thể thực hiện (ví dụ: gửi tiền, rút tiền).
  • Danh tính (Identity): mỗi đối tượng có một ID duy nhất để phân biệt với các đối tượng khác trong JVM.

Ví dụ: Một cây bút chì có tên là A, màu trắng là trạng thái, còn việc sử dụng bút để viết là hành vi.

Lưu ý: Đối tượng là thể hiện (instance) của một lớp.

2. Lớp (Class)

Một lớp là khuôn mẫu hoặc thiết kế từ đó các đối tượng được tạo ra. Một lớp có thể chứa:

  • Thành viên dữ liệu (fields)
  • Constructor (hàm khởi tạo)
  • Phương thức (methods)
  • Khối lệnh (blocks)
  • Interface (nếu cần)

3. Ví dụ về lớp và đối tượng

Ví dụ 1: Lớp Student với dữ liệu mặc định

public class Student {
    int id; // thành viên dữ liệu
    String name; // thành viên dữ liệu

    public static void main(String args[]) {
        Student student1 = new Student(); // tạo đối tượng student1
        System.out.println(student1.id);
        System.out.println(student1.name);
    }
}

Kết quả:
0
null

Ví dụ 2: Lớp Student2 với phương thức

public class Student2 {
    int id;
    String name;

    void insertRecord(int id, String name) { 
        this.id = id;
        this.name = name;
    }

    void displayInformation() {
        System.out.println(id + " " + name);
    }

    public static void main(String args[]) {
        Student2 s1 = new Student2();
        Student2 s2 = new Student2();

        s1.insertRecord(111, "Viet");
        s2.insertRecord(222, "Tuts");

        s1.displayInformation();
        s2.displayInformation();
    }
}

Kết quả:
111 Viet
222 Tuts

Ví dụ 3: Lớp Student3 với Constructor

4. Các cách tạo đối tượng trong Java

Sử dụng từ khóa new

Sử dụng phương thức newInstance()

Sử dụng phương thức clone()

Sử dụng phương thức factory

Các cách tạo đối tượng nâng cao sẽ được học trong các bài sau.

5. Đối tượng Annonymous (Vô danh)

Annonymous nghĩa là vô danh – đối tượng không có tham chiếu. Thường dùng khi chỉ cần sử dụng đối tượng một lần duy nhất.

public class Calculation {
    void fact(int n) {
        int giaithua = 1;
        for (int i = 1; i <= n; i++) {
            giaithua = giaithua * i;
        }
        System.out.println("Giai thừa của " + n + " là: " + giaithua);
    }

    public static void main(String args[]) {
        new Calculation().fact(5); // đối tượng annonymous
    }
}

Kết quả:

Giai thừa của 5 là: 120

6. Sự khác nhau giữa lớp và đối tượng

No Đối tượng Lớp
1 Thể hiện của 1 lớp Khuôn mẫu để tạo đối tượng
2 Thực thể trong thế giới thực (bút, xe đạp…) Nhóm các đối tượng tương tự
3 Thực thể vật lý Thực thể logic
4 Tạo bằng new (ví dụ: Student s1 = new Student();) Khai báo bằng class (ví dụ: class Student {})
5 Có thể tạo nhiều lần Được khai báo 1 lần duy nhất
6 Được cấp bộ nhớ khi tạo Không cấp bộ nhớ khi khai báo
7 Nhiều cách tạo: new, newInstance(), clone(), factory, deserialization Chỉ một cách: sử dụng class
