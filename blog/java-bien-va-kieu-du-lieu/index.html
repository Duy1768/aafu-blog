Biến và kiểu dữ liệu trong Java

Trong java, biến là tên của vùng nhớ được lưu trong bộ nhớ stack. Có 3 kiểu biến trong java, bao gồm biến cục bộ (local), biến toàn cục (instance) và biến tĩnh (static).

1. Biến local trong Java

  • Khai báo trong phương thức, constructor hoặc block
  • Tồn tại trong phạm vi phương thức/block
  • Không dùng access modifier
  • Nằm trên stack
public void sayHello() {
    int n = 10;
    System.out.println("Gia tri cua n la: " + n);
}

Chú ý: Nếu không khởi tạo giá trị, biến local sẽ báo lỗi khi biên dịch.

2. Biến instance

  • Khai báo trong class, ngoài phương thức

  • Lưu trong heap

  • Tạo khi đối tượng được tạo

  • Có giá trị mặc định theo kiểu dữ liệu

public class Sinhvien {
    public String ten;
    private int tuoi;

    public Sinhvien(String ten) {
        this.ten = ten;
    }

    public void setTuoi(int tuoi) {
        this.tuoi = tuoi;
    }
    
    public void showStudent() {
        System.out.println("Ten  : " + ten);
        System.out.println("Tuoi : " + tuoi);
    }
}

Kết quả:

Ten  : Nguyen Van A
Tuoi : 21

3. Biến static

  • Khai báo với từ khóa static

  • Chỉ có một bản sao

  • Lưu trong bộ nhớ static

  • Tạo khi chương trình chạy

public class Sinhvien {
    public static String ten = "Nguyen Van A";
    public static int tuoi = 21;

    public static void main(String args[]) {
        System.out.println("Ten : " + ten);
        System.out.println("Tuoi : " + Sinhvien.tuoi);
    }
}

Kết quả:

Ten  : Nguyen Van A
Tuoi : 21

4. Các kiểu dữ liệu

a. Nguyên thủy (Primitive)

Kiểu Mô tả Giá trị mặc định
byte Số nguyên 1 byte 0
char Kí tự 2 byte 0
boolean True/False False
short Số nguyên 2 byte 0
int Số nguyên 4 byte 0
long Số nguyên 8 byte 0L
float Số thực 4 byte 0.0F
double Số thực 8 byte 0.0D

b. Đối tượng (Object)

Array: mảng dữ liệu cùng kiểu

class: lớp do người dùng định nghĩa, chứa thuộc tính & phương thức

interface: giao tiếp, chứa phương thức mà lớp implement phải định nghĩa

5. Tổng kết

Biến local: phạm vi phương thức, stack, phải khởi tạo trước khi dùng

Biến instance: thuộc đối tượng, lưu heap, có giá trị mặc định

Biến static: thuộc class, lưu static memory, chỉ một bản sao

Kiểu dữ liệu: nguyên thủy và đối tượng
